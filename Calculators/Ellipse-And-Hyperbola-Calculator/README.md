# <p align="center">Ellipse And Hyperbola Calculator</p>

## Description :-

This is a simple web-based calculator that allows users to calculate the area and, where applicable, the perimeter of ellipses and hyperbolas.

## Tech Stacks :-

- HTML
- CSS
- JavaScript
