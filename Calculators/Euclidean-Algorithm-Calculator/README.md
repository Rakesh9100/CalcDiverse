# Euclidean Algorithm Calculator

### Functionality :-
This calculator is a simple tool that to compute the Greatest Common Divisor (GCD) and the Least Common Multiple (LCM) of two integers entered by the user.


### Built Using :-

- HTML
- CSS
- JavaScript
