# <p align="center">Euclidean Algorithm Calculator</p>

## Description :-

This calculator is a simple tool that computes the Greatest Common Divisor (GCD) and the Least Common Multiple (LCM) of two integers entered by the user.

## Tech Stacks :-

- HTML
- CSS
- JavaScript
