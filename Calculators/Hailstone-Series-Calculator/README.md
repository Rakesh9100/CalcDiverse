# <p align="center">Hailstone Series Calculator</p>

## Description :-

A Hailstone series starts with a positive integer; if the number is even, divide it by 2, if odd, multiply by 3 and add 1, and repeat.

## Tech Stacks :-

- HTML
- CSS
