# <p align="center">Harmonic Progression Calculator</p>

## Description :-

A simple web-based calculator to find the nth term, sum of first n terms and harmonic mean of a Harmonic Progression (HP).

## Tech Stacks :-

- HTML
- CSS
- JavaScript
