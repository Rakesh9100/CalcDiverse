# <p align="center">Isogram Calculator</p>

## Description :-

A simple web-based calculator to find whether the string is Isogram or not.<br>
**Calculator logic:** An isogram is a word in which no letter occurs more than once.

## Tech Stacks :-

- HTML
- CSS
- JavaScript
