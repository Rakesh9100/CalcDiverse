# <p align="center">Midpoint Calculator</p>

## Description :-

A web-based midpoint calculator where there are 4 input fields consisting of x1, x2, y1, y2.
Users have to fill in this value to get mid point.

## Tech Stacks :-

- HTML
- CSS
- JavaScript
