# <p align="center">Planetary Motion Calculator</p>

## Description :-

This calculator calculates orbital time period, semi-major, semi-minor axis lengths, and areal velocity using Kepler's laws.

## Tech Stacks :-

- HTML
- CSS
- JavaScript
