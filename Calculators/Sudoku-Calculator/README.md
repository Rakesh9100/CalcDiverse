# <p align="center">Sudoku Calculator</p>

## Description :-

- Generates and Calculates Sudoku
- Can solve user given sudoko's as well

## Tech Stacks :-

- HTML
- CSS
- JavaScript
- ConfettiJS Library for confetti effect after solving
