# <p align="center">Trigonometric Function Graphing Calculator</p>

## Description :-

This is a Trigonometric Function Graphing Calculator which gives the solution with a graph as per the given function.

For example -
- sin
- cos
- tan, etc.

Example - 

Math.sin(x/30)*100

## Tech Stack:-

- HTML
- CSS
- JavaScript
