# <p align="center">Trisection Point Calculator</p>

## Description :-

A web-based Trisection point calculator where there are 4 input fields consisting of x1, x2, y1, y2.
Users have to fill in these fields to get Trisection points. 

## Tech Stacks :-

- HTML
- CSS
- JavaScript
