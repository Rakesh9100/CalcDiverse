# <p align="center">Wavelength Calculator</p>

## Description :-

The Wavelength Calculator is a simple web application that allows users to calculate the wavelength of a wave using the formula:

λ = v/f  
where:  
λ -> wavelength  
v-> velocity  
f -> frequency  

## Tech Stacks :-

- HTML
- CSS
- JavaScript
